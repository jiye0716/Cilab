# Git

## git config --global user.name "xxxxx"

## git config --global user.email xxxxxxx@example.com

## git init

    initial初始化

## git pull

    從遠端拉取回本地（同步）

## git clone

    從遠端複製資料

## git status

    顯示修改檔案清單
    -s：僅會顯示已修改的檔案名稱

## git add .

    將子目錄裡的所有檔案註冊到索引裡

## git commit

    -a : 有修改的檔案(不包括新增的檔案)，將其加入索引並提交。
    -m : 提交訊息

## git push

    從本地推送到遠端
    git push origin master 本地master分支推一份到origin節點

## git rm

    remove
